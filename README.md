# CRUD_List

This code manipulates lists of movies. 

## Options

```
[1] - Add movie...
[2] - Search movie...
[3] - Modify movie...
[4] - Delete movie...
[5] - Show list...
[6] - Show by genre...
[7] - Export file to HTML...
[8] - Exit...
```

## Run

To execute the code use the option of c++ 11: **-std=c++11**. 
*Observation*: in CodeBlocks this option can be enabled in Settings/Compiler.
